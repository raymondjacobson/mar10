## How to contribute

1. Fork the repo and create a branch from `main`
2. Make your changes
3. Open a pull request with a clear description of what you changed and why
4. PRs that break the game or remove the Audius music integration will not be merged

## Local development

```bash
# Serve the static files locally (any static server works)
npx serve .
# or
python3 -m http.server 8080
```

Then open `http://localhost:8080` in your browser.
