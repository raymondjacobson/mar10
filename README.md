# mar10

A Super Mario Bros clone with purple Mario and background music streamed live from [Audius](https://audius.co).

**Play it:** https://mar10.vercel.app

## What's different

- Mario's red recolored to purple `#7f6ad6` via canvas pixel manipulation at load time
- Background music sourced from Audius — searches for `nintendo` tracks, shuffles them, and auto-queues the next on end
- Now-playing bar shows the current track; click it to open it on Audius

## Controls

| Action | Key |
|--------|-----|
| Move   | Arrow keys |
| Jump   | X |
| Run    | Z |
